0 文章说明

本文使用 Python 完成一个完整的小型数据分析案例。分析对象是一组“学习行为数据”，每一行代表一名同学的学习情况，变量包括学习时长、出勤率、复习次数、练习题数量、课堂参与度和最终成绩。

这篇文章的写法不是简单堆代码，而是按照一篇数据分析报告的逻辑展开：先说明数据来源和变量含义，再进行数据查看与清洗，然后通过描述统计和图形观察数据特征，最后建立回归模型，分析学习行为对成绩的影响。

1 导入分析包

首先导入本次分析需要用到的 Python 包。pandas 负责处理表格数据，numpy 负责生成模拟数据和进行数值运算，matplotlib 用来绘图，scikit-learn 用来建立线性回归模型。

为了让图表中的中文正常显示，需要设置 matplotlib 的中文字体。如果电脑中没有对应字体，图形仍然可以生成，只是中文可能显示为方框。

2 构造学习行为数据集

为了让文章可以直接运行，这里不从外部文件读取数据，而是在代码中构造一个模拟数据集。这个数据集虽然是模拟的，但变量设计尽量贴近真实学习场景。

数据集中包含以下变量：

变量名 含义
student_id 学生编号
group 学习小组
study_hours 每日平均学习时长
attendance 出勤率
review_times 每周复习次数
practice_count 每周练习题数量
participation 课堂参与度
score 期末成绩
代码 
n = 80

student_id = [f"S{i:03d}" for i in range(1, n + 1)]
group = np.random.choice(["A组", "B组", "C组", "D组"], size=n, p=[0.25, 0.25, 0.25, 0.25])

study_hours = np.random.normal(loc=4.2, scale=1.1, size=n)
study_hours = np.clip(study_hours, 1.2, 7.5)

attendance = np.random.normal(loc=0.88, scale=0.07, size=n)
attendance = np.clip(attendance, 0.60, 1.00)

review_times = np.random.poisson(lam=3.2, size=n)
review_times = np.clip(review_times, 0, 8)

practice_count = np.random.normal(loc=38, scale=12, size=n)
practice_count = np.clip(practice_count, 8, 75).round()

participation = np.random.normal(loc=75, scale=12, size=n)
participation = np.clip(participation, 40, 100)

noise = np.random.normal(loc=0, scale=4.5, size=n)

score = (
    35
    + study_hours * 5.8
    + attendance * 16
    + review_times * 1.3
    + practice_count * 0.22
    + participation * 0.12
    + noise
)

score = np.clip(score, 45, 100).round(1)

df = pd.DataFrame({
    "student_id": student_id,
    "group": group,
    "study_hours": study_hours.round(2),
    "attendance": attendance.round(3),
    "review_times": review_times,
    "practice_count": practice_count.astype(int),
    "participation": participation.round(1),
    "score": score
})

df.loc[[8, 19, 37], "attendance"] = np.nan
df.loc[[14, 46], "review_times"] = np.nan
df.loc[[25, 62], "practice_count"] = np.nan

print("数据集构造完成。")
print(f"样本数量：{df.shape[0]}")
print(f"变量数量：{df.shape[1]}")

4 缺失值处理

真实数据中经常存在缺失值。比如有的同学没有填写出勤率，有的同学没有记录复习次数，还有的同学缺少练习题数量。如果直接忽略这些缺失值，可能会造成样本减少；如果处理不当，也可能影响模型结果。

先统计每个变量的缺失情况。

代码 
missing_table = pd.DataFrame({
    "缺失个数": df.isna().sum(),
    "缺失比例": (df.isna().mean() * 100).round(2)
})

可以看到，缺失值主要集中在 attendance、review_times 和 practice_count 这几个变量上。由于缺失比例较低，本文采用中位数填补。中位数填补的优点是简单、稳定，不容易受到极端值影响。

代码 
df_clean = df.copy()

for col in ["attendance", "review_times", "practice_count"]:
    median_value = df_clean[col].median()
    df_clean[col] = df_clean[col].fillna(median_value)

print("缺失值填补完成。")
print("填补后各变量缺失值数量：")
print(df_clean.isna().sum())

5 描述性统计

描述性统计用于快速了解数据的整体特征。这里主要观察均值、标准差、最小值、最大值和四分位数。

如果一个变量的标准差很大，说明不同学生之间差异明显；如果最大值和最小值差距很大，说明该变量的分布范围较宽。

6 分组比较

除了整体描述统计，还可以按照学习小组进行分组比较。分组比较可以帮助我们观察不同小组之间是否存在明显差异。

这里计算每个小组的平均学习时长、平均练习题数量、平均课堂参与度和平均成绩。

代码 
group_summary = (

        平均练习题数量=("practice_count", "mean"),
        平均参与度=("participation", "mean"),

8 相关性分析

相关性分析用于衡量两个数值变量之间的线性关系。相关系数的取值范围在 -1 到 1 之间：

接近 -1 强负相关

9 回归建模

相关性分析只能说明变量之间是否存在线性关系，而回归模型可以进一步用于预测。这里使用多元线性回归模型，将学习行为变量作为自变量，将成绩作为因变量。

模型形式可以写成：

[ score = _0 + _1 study_hours + _2 attendance + _3 review_times + _4 practice_count + _5 participation + ]

其中，(_0) 是截距项，(_1,_2,,_5) 是各变量对应的回归系数。

代码 
features = ["study_hours", "attendance", "review_times", "practice_count", "participation"]
target = "score"

X = df_clean[features]
y = df_clean[target]

X_train, X_test, y_train, y_test = train_test_split(
    X, y,
    test_size=0.25,
    random_state=20260703
)

model = LinearRegression()
model.fit(X_train, y_train)

coef_table = pd.DataFrame({
    "变量": features,
    "回归系数": model.coef_.round(3)
})

intercept = round(model.intercept_, 3)

print(f"模型截距：{intercept}")
coef_table
模型截距：50.26
变量 回归系数
0 study_hours 4.123
1 attendance 14.588
2 review_times 1.038
3 practice_count 0.096
4 participation 0.089

回归系数可以帮助我们理解每个变量对成绩预测的贡献方向。系数为正，说明该变量增加时，预测成绩倾向于提高；系数为负，则说明该变量增加时，预测成绩倾向于下降。

需要注意的是，回归系数并不一定等于因果效应。尤其是在模拟数据之外的真实数据中，还可能存在遗漏变量、样本偏差等问题。

11 模型评价

为了更准确地评价模型，需要计算几个常见指标：

指标 含义
MAE 平均绝对误差，数值越小越好
RMSE 均方根误差，对较大误差更敏感
R² 拟合优度，越接近 1 表示解释能力越强
代码 
mae = mean_absolute_error(y_test, y_pred)
rmse = np.sqrt(mean_squared_error(y_test, y_pred))
r2 = r2_score(y_test, y_pred)

evaluation = pd.DataFrame({
    "指标": ["MAE", "RMSE", "R²"],
    "数值": [round(mae, 3), round(rmse, 3), round(r2, 3)]
})

evaluation
指标 数值
0 MAE 3.493
1 RMSE 4.230
2 R² 0.700

如果 MAE 和 RMSE 较低，说明模型预测误差较小；如果 R² 较高，说明模型能够解释较多的成绩差异。

当然，这只是一个小型示例。真实研究中还需要考虑样本量、变量选择、模型假设、异常值、过拟合等问题。

12 结论

本文使用 pandas、numpy、matplotlib 和 scikit-learn 完成了一篇完整的 Python 数据分析文章。整个流程可以概括为：

  1. 先构造并查看数据，明确每个变量的含义；
  2. 再检查缺失值，并使用中位数进行填补；
  3. 接着通过描述统计了解数据整体情况；
  4. 然后使用柱状图、直方图、散点图和热力图观察变量关系；
  5. 最后建立多元线性回归模型，对学生成绩进行预测，并使用 MAE、RMSE 和 R² 评价模型效果。

这篇文章的重点不是展示复杂模型，而是展示一篇博客型数据分析文章应该如何组织：先讲清楚问题，再展示代码，然后解释输出结果。这样读者不仅能看到代码怎么写，也能理解每一步分析为什么要这样做。

# 2 构造学习行为数据集 {#data}

为了让文章可以直接运行，这里不从外部文件读取数据，而是在代码中构造一个模拟数据集。这个数据集虽然是模拟的，但变量设计尽量贴近真实学习场景。

数据集中包含以下变量：

| 变量名 | 含义 |
|---|---|
| `student_id` | 学生编号 |
| `group` | 学习小组 |
| `study_hours` | 每日平均学习时长 |
| `attendance` | 出勤率 |
| `review_times` | 每周复习次数 |
| `practice_count` | 每周练习题数量 |
| `participation` | 课堂参与度 |
| `score` | 期末成绩 |

```{python}
#| label: build-data
#| code-summary: "代码"

n = 80

student_id = [f"S{i:03d}" for i in range(1, n + 1)]
group = np.random.choice(["A组", "B组", "C组", "D组"], size=n, p=[0.25, 0.25, 0.25, 0.25])

study_hours = np.random.normal(loc=4.2, scale=1.1, size=n)
study_hours = np.clip(study_hours, 1.2, 7.5)

attendance = np.random.normal(loc=0.88, scale=0.07, size=n)
attendance = np.clip(attendance, 0.60, 1.00)

review_times = np.random.poisson(lam=3.2, size=n)
review_times = np.clip(review_times, 0, 8)

practice_count = np.random.normal(loc=38, scale=12, size=n)
practice_count = np.clip(practice_count, 8, 75).round()

participation = np.random.normal(loc=75, scale=12, size=n)
participation = np.clip(participation, 40, 100)

noise = np.random.normal(loc=0, scale=4.5, size=n)

score = (
    35
    + study_hours * 5.8
    + attendance * 16
    + review_times * 1.3
    + practice_count * 0.22
    + participation * 0.12
    + noise
)

score = np.clip(score, 45, 100).round(1)

df = pd.DataFrame({
    "student_id": student_id,
    "group": group,
    "study_hours": study_hours.round(2),
    "attendance": attendance.round(3),
    "review_times": review_times,
    "practice_count": practice_count.astype(int),
    "participation": participation.round(1),
    "score": score
})

df.loc[[8, 19, 37], "attendance"] = np.nan
df.loc[[14, 46], "review_times"] = np.nan
df.loc[[25, 62], "practice_count"] = np.nan

print("数据集构造完成。")
print(f"样本数量：{df.shape[0]}")
print(f"变量数量：{df.shape[1]}")
```

# 4 缺失值处理 {#missing}

真实数据中经常存在缺失值。比如有的同学没有填写出勤率，有的同学没有记录复习次数，还有的同学缺少练习题数量。如果直接忽略这些缺失值，可能会造成样本减少；如果处理不当，也可能影响模型结果。

先统计每个变量的缺失情况。

```{python}
#| label: missing-summary
#| code-summary: "代码"

missing_table = pd.DataFrame({
    "缺失个数": df.isna().sum(),
    "缺失比例": (df.isna().mean() * 100).round(2)
})

missing_table
```

可以看到，缺失值主要集中在 `attendance`、`review_times` 和 `practice_count` 这几个变量上。由于缺失比例较低，本文采用中位数填补。中位数填补的优点是简单、稳定，不容易受到极端值影响。

```{python}
#| label: missing-fill
#| code-summary: "代码"

df_clean = df.copy()

for col in ["attendance", "review_times", "practice_count"]:
    median_value = df_clean[col].median()
    df_clean[col] = df_clean[col].fillna(median_value)

print("缺失值填补完成。")
print("填补后各变量缺失值数量：")
print(df_clean.isna().sum())
```

desc

group_summary = (

# 8 相关性分析 {#corr}

相关性分析用于衡量两个数值变量之间的线性关系。相关系数的取值范围在 -1 到 1 之间：

| 相关系数 | 含义 |
|---|---|
| 接近 1 | 强正相关 |
| 接近 -1 | 强负相关 |
| 接近 0 | 线性关系较弱 |

下面计算各学习行为变量与成绩之间的相关系数。

```{python}
#| label: correlation-table
#| code-summary: "代码"

corr = df_clean[num_cols].corr().round(3)

score_corr = (
    corr["score"]
    .drop("score")
    .sort_values(ascending=False)
    .to_frame(name="与成绩的相关系数")
)

score_corr
```

为了更直观地观察变量之间的关系，可以绘制相关性热力图。颜色越深，说明相关程度越强。

```{python}
#| label: correlation-heatmap
#| code-summary: "代码"

plt.figure(figsize=(8, 6))
plt.imshow(corr, aspect="auto")

plt.tight_layout()
plt.show()
```

从相关性分析中可以判断，哪些学习行为变量和成绩关系更紧密。一般来说，学习时长、练习题数量和课堂参与度都可能对成绩有较明显影响。

# 9 回归建模 {#regression}

相关性分析只能说明变量之间是否存在线性关系，而回归模型可以进一步用于预测。这里使用多元线性回归模型，将学习行为变量作为自变量，将成绩作为因变量。

模型形式可以写成：

\[
score = \beta_0 + \beta_1 study\_hours + \beta_2 attendance + \beta_3 review\_times + \beta_4 practice\_count + \beta_5 participation + \varepsilon
\]

其中，\(\beta_0\) 是截距项，\(\beta_1,\beta_2,\ldots,\beta_5\) 是各变量对应的回归系数。

```{python}
#| label: regression-model
#| code-summary: "代码"

features = ["study_hours", "attendance", "review_times", "practice_count", "participation"]
target = "score"

X = df_clean[features]
y = df_clean[target]

X_train, X_test, y_train, y_test = train_test_split(
    X, y,
    test_size=0.25,
    random_state=20260703
)

model = LinearRegression()
model.fit(X_train, y_train)

coef_table = pd.DataFrame({
    "变量": features,
    "回归系数": model.coef_.round(3)
})

intercept = round(model.intercept_, 3)

print(f"模型截距：{intercept}")
coef_table
```

回归系数可以帮助我们理解每个变量对成绩预测的贡献方向。系数为正，说明该变量增加时，预测成绩倾向于提高；系数为负，则说明该变量增加时，预测成绩倾向于下降。

需要注意的是，回归系数并不一定等于因果效应。尤其是在模拟数据之外的真实数据中，还可能存在遗漏变量、样本偏差等问题。

# 10 预测结果 {#prediction}

模型训练完成后，可以使用测试集检验预测效果。测试集中的数据没有参与模型训练，因此更适合用来观察模型的泛化能力。

```{python}
#| label: prediction
#| code-summary: "代码"

y_pred = model.predict(X_test)

pred_table = pd.DataFrame({
    "实际成绩": y_test.values,
    "预测成绩": y_pred.round(1),
    "预测误差": (y_test.values - y_pred).round(1)
})

pred_table.head(12)
```

下面绘制“实际成绩 vs 预测成绩”的散点图。如果模型预测较好，点应该大致分布在对角线附近。

```{python}
#| label: prediction-plot
#| code-summary: "代码"

plt.figure(figsize=(7, 6))
plt.scatter(y_test, y_pred, alpha=0.75)

min_value = min(y_test.min(), y_pred.min())
max_value = max(y_test.max(), y_pred.max())

plt.plot([min_value, max_value], [min_value, max_value], linestyle="--")
plt.title("实际成绩与预测成绩对比")
plt.xlabel("实际成绩")
plt.ylabel("预测成绩")
plt.grid(alpha=0.25)
plt.show()
```

图中虚线表示理想情况，也就是预测值等于真实值。散点越靠近虚线，说明模型预测越准确；散点偏离越大，说明预测误差越大。

# 11 模型评价 {#evaluation}

为了更准确地评价模型，需要计算几个常见指标：

| 指标 | 含义 |
|---|---|
| MAE | 平均绝对误差，数值越小越好 |
| RMSE | 均方根误差，对较大误差更敏感 |
| R² | 拟合优度，越接近 1 表示解释能力越强 |

```{python}
#| label: model-evaluation
#| code-summary: "代码"

mae = mean_absolute_error(y_test, y_pred)
rmse = np.sqrt(mean_squared_error(y_test, y_pred))
r2 = r2_score(y_test, y_pred)

evaluation = pd.DataFrame({
    "指标": ["MAE", "RMSE", "R²"],
    "数值": [round(mae, 3), round(rmse, 3), round(r2, 3)]
})

evaluation
```

如果 MAE 和 RMSE 较低，说明模型预测误差较小；如果 R² 较高，说明模型能够解释较多的成绩差异。

当然，这只是一个小型示例。真实研究中还需要考虑样本量、变量选择、模型假设、异常值、过拟合等问题。
